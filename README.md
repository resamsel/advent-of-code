🎄 Advent of Code 🎄
====================

Solutions to [Advent of Code](https://adventofcode.com/) puzzles written in TypeScript.

## 📥 Setup

```shell
npm install
```

## 🏃 Run

Load the project in your favourite IDE and execute the matching test. Alternatively you can run the tests on the CLI:

```shell
npm run test
```
